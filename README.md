## Installing - 安裝流程

1. 開啟終端機(terminal)並複製(clone)此專案至本機電腦
```
git link here
```
2. 在終端機(terminal)中進入此專案資料夾
```
cd todo_list
```
3. 安裝npm套件以及nodemon套件
```
npm install // 安裝npm套件
npm install -g nodemon // 安裝nodemon套件
```
4. 執行app.js啟動伺服器
```
npm run dev
```
5. 開啟瀏覽器輸入 http://localhost:3000 即可進入Todo List
